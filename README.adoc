= Nyra

An open-source Discord bot built with *Discord.Net*.

---

== Why Nyra?

This project exists for two main reasons:

1. To refresh my C# skills after a ~5–6 year break on it.
2. To evaluate whether C# is a language I’d enjoy working with again.
