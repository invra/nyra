= Nyra

An open-source Discord bot built with Rust and *poise*.

There is a GUI written GUI Zed Editor's *GPUI* framework

---

== Why Nyra?

This project exists for two main reasons:

1. So I can learn more stuff about Rust
2. When it was written with C# and Rust, it was about Interop.
